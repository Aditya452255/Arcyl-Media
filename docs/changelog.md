# Changelog - Arcyl Media

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-28
### Added
- Phase 1 documentation: roadmap, folder structure, database, api-design, and architecture.
- Modular architectural pattern implementation (Repository, Service, Controller).
- Zod schema validations for public forms.
- Centralized errors utility and Next.js route wrapper.
- Resend, Cloudinary, and Pino logger configurations.
- API Route `/api/contact` recording leads, submissions, and auditing trails.
