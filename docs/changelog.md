# Changelog - Arcyl Media

All notable changes to this project will be documented in this file.

## [2.1.0] - 2026-06-28
### Added
- **Phase 6.5 Business Essentials**: Quick quotations converter, automatic serial Proposal/Invoice numbers, client timeline logs, email templates via Resend, and browser Print-to-PDF styles.
- **Phase 6 Client Portal**: Secure tenant-isolated login portal, read-only tasks, deliverable reviews, project message boards, and shared document uploads.
- **Phase 5 Tasks Board**: Interactive task cards Kanban board, drag-and-drop lists, and automated Milestone and Project progress hook calculations.
- **Phase 4 Workspace**: Projects lifecycle milestones roadmap, shared files ledgers, deliverable pipeline versioning, and client details cards.
- **Phase 3 Employee Operations**: Employee directories, department registers, and activity log tracking.

## [2.0.0] - 2026-06-28
### Added
- Phase 2 features: Admin Dashboard UI, dynamic CMS contents, Cloudinary media library registers, JWT Authentication, and RBAC customs permissions matrices.

## [1.0.0] - 2026-06-28
### Added
- Phase 1 core foundation: Next.js 15, Prisma db setup, central errors utility, Zod validators, Resend client, and Contact API.
